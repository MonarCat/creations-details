# Creations & Details Limited Website

Multi-page static website for Creations & Details Limited (Nairobi-based interior design and construction company), structured for easy deployment to Netlify or Vercel.

## Folder Structure

- `index.html`, `about.html`, `services.html`, `portfolio.html`, `team.html`, `contact.html`
- `assets/css/` for global and page-level styles
- `assets/js/` for global and page-level scripts
- `assets/images/projects/*` and `assets/images/team/` for image placeholders
- `vercel.json` for deployment headers and URL behavior

## Run Locally

No build step is required.

1. Open `index.html` directly in a browser.
2. Navigate through the pages using the top menu.

## Deploy to Vercel

- Drag and drop the repository directory in the Vercel dashboard, **or**
- Run `vercel --prod` from this repository root.
